# Unit 1 Practicals
# DevOps – Unit 1

## Overview

This unit introduces the basic concepts of DevOps and containerization. It includes installation and configuration of Docker, understanding Docker commands, creating containers, and working with Docker images.

---

# Practical 01

## Aim

Install Docker and verify the Docker installation.

## Steps

1. Install Docker on the system.
2. Start the Docker service.
3. Check Docker version.
4. Run a test container to verify the installation.

## Result

Docker was successfully installed and verified using Docker commands.

---

# Practical 02

## Aim

Demonstrate basic Docker commands and container management.

## Steps

1. Pull a Docker image from Docker Hub.
2. Run a container using the image.
3. List running containers.
4. Stop and remove containers.
5. Remove Docker images.

## Result

Docker containers were successfully created, managed, and removed using Docker commands.
